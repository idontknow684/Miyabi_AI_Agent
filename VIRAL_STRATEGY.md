## 🎯 バイラルの核心コンセプト

### キャッチフレーズ

**"5分でAIエージェントを作れる世界"**

```
コードゼロ → 動くエージェント
専門知識不要 → すぐに開始
複雑な理論 → シンプルなテンプレート
孤独な開発 → 活発なコミュニティ
```

### バイラルフック（3つの驚き）

1. **"えっ、コード書かなくていいの？"**
   - `.claude/agents/`にMarkdownファイルを置くだけ
   - すぐに動作するエージェント

2. **"うわ、こんなに簡単なの？"**
   - `Use this template` → 即開始
   - 5分でカスタマイズ完了

3. **"まじで？これ無料？"**
   - 完全オープンソース
   - 61,251行のドキュメント込み
   - 学術論文まで付属

## 🎯 低敷居化戦略

### 初心者が躓くポイント → 解決策

| 躓きポイント | 従来 | このテンプレート |
|------------|------|----------------|
| **環境構築** | 複雑な設定 | `Use this template` 1クリック |
| **コード記述** | Rustを学ぶ必要 | Markdownで定義 |
| **理論理解** | 難解な論文 | 図解付きドキュメント |
| **質問相手** | Stack Overflow待ち | Discord即回答 |
| **デバッグ** | 孤独な戦い | コミュニティサポート |
| **事例不足** | 自分で考える | Showcase Gallery |

### 3段階オンボーディング

#### Level 1: 超初心者（コードゼロ）

**目標**: 5分で動くものを作る

**ステップ:**
1. `Use this template` クリック
2. README.mdのプロジェクト名変更
3. `.claude/agents/hello.md` 作成
```markdown
---
name: hello
description: Hello world agent
---

# Hello Agent
Say hello!
```
4. 完成！

**成果**: 動くエージェント、達成感

#### Level 2: 初心者（カスタマイズ）

**目標**: 30分で自分好みにカスタマイズ

**ステップ:**
1. カスタムエージェント追加
2. ドキュメント編集
3. PlantUML図追加
4. ビジネス計画カスタマイズ

**成果**: オリジナルプロジェクト、自信

#### Level 3: 中級者（実装開始）

**目標**: 1週間でRust実装スタート

**ステップ:**
1. `cargo new crates/my-core`
2. 基本構造実装
3. テスト追加
4. ドキュメント更新

**成果**: 動作するプロダクト、実力

## Tier 1: Strategic Partnerships

### 1. AWS Partnership

**Rationale**:
- 83% of enterprises use AWS
- AWS Marketplace = trusted procurement channel
- Co-selling = access to AWS sales reps

**Partnership Structure**:

#### AWS Marketplace Listing (Priority: P0, Timeline: Q1 2026)

**Benefits**:
- **Procurement Simplification**: Enterprise customers can purchase through existing AWS billing
- **Credibility**: AWS vets all marketplace apps (security, reliability)
- **Discovery**: 500K+ monthly visitors to AWS Marketplace
- **Co-Selling**: AWS sales reps can recommend Miyabi (20% commission to AWS)

**Implementation**:
1. **Week 1-2**: AWS ISV Onboarding Program application
2. **Week 3-4**: Security review (AWS vets codebase, pen testing)
3. **Week 5-6**: Listing creation (product page, pricing, docs)
4. **Week 7-8**: Go-live (publish to marketplace)

**Pricing on AWS Marketplace**:
- Same as direct: $299/seat/month (Enterprise tier)
- Revenue share: 80% Miyabi / 20% AWS (industry standard)
- Payment terms: AWS pays Miyabi monthly (even if customer pays annually)

---

#### AWS Co-Sell Program (Priority: P0, Timeline: Q2 2026)

**Benefits**:
- **Sales Acceleration**: AWS Enterprise Sales reps recommend Miyabi to their accounts
- **Commission**: AWS reps earn 10-20% commission on Miyabi deals
- **Deal Registration**: AWS shares active opportunities (e.g., "Bank of America is modernizing DevOps")
- **Joint Pursuit**: AWS + Miyabi collaborate on large deals (>$1M)

**Requirements**:
- ✅ AWS Marketplace listing (prerequisite)
- ✅ 5+ joint customer references
- ✅ AWS Competency (DevOps or Machine Learning)
- ✅ Dedicated AWS Alliance Manager (hire Q1 2026)

**Implementation**:
1. **Q1 2026**: Hire AWS Alliance Manager ($150K base + $50K variable)
2. **Q2 2026**: Apply for AWS DevOps Competency (security audit, case studies)
3. **Q2 2026**: Launch co-sell program (train 50 AWS reps on Miyabi)
4. **Q3 2026**: First joint customer win (target: $500K+ deal)

**Target**: 10 co-sell deals by Q4 2026 (average $200K ACV = $2M ARR)

---

#### AWS for Startups Program (Priority: P1, Timeline: Q1 2026)

**Benefits**:
- **Free AWS Credits**: $100K in AWS credits for Miyabi infrastructure
- **Technical Support**: AWS Solutions Architects assist with architecture
- **Marketing**: Featured in AWS Startups blog, social media

**Requirements**:
- ✅ Series A funded startup (we have $5M seed)
- ✅ Building on AWS (Miyabi infrastructure is AWS-native)
- ✅ High-growth potential (targeting $100M ARR by 2028)

**Implementation**:
1. Apply via AWS for Startups portal
2. AWS vets application (2 weeks)

### 3. KDDI Partnership (Japan Market Entry)

**Rationale**:
- KDDI is Japan's 2nd largest telco (50M+ subscribers)
- KDDI has enterprise IT services division (sells to Fortune 500 Japan)
- KDDI can provide local sales, support, and cultural fit

**Partnership Structure**:

#### Reseller Agreement (Priority: P0, Timeline: Q2 2026)

**Terms**:
- **Revenue Share**: 70% Miyabi / 30% KDDI (KDDI handles sales, support)
- **Exclusivity**: KDDI is exclusive reseller in Japan (Miyabi can still sell direct to global accounts with Japan presence)
- **Minimum Commitment**: $1M ARR in Year 1 (KDDI at risk if not hit)

**Why KDDI Wins**:
- **Local Sales Team**: 200+ enterprise sales reps (sell Miyabi alongside other KDDI services)
- **Trusted Brand**: KDDI is household name in Japan (enterprise credibility)
- **Cultural Fit**: KDDI understands Japanese business culture (long sales cycles, relationship-driven)
- **Support**: KDDI provides Japanese-language support (24/7)

**Implementation**:
1. **Q1 2026**: Identify KDDI business development contact (warm intro via investor)
2. **Q2 2026**: Negotiate reseller agreement (legal, pricing, support SLA)
3. **Q2 2026**: KDDI sales training (2-day workshop, train 50 reps)
4. **Q3 2026**: Co-marketing launch (press release, case study)
5. **Q4 2026**: First KDDI-sourced customer

**Target**: 5 enterprise deals by Q4 2026 (average $100K ACV = $500K ARR)
By Q4 2027: 20 deals ($2M ARR)

---

#### KDDI Co-Marketing

## üí° Pro Tips

### For New Sales Reps

**Week 1**:
- Focus on learning the product (demos, demos, demos)
- Don't try to sell yet - just learn

**Week 2-3**:
- Shadow experienced reps (listen to 10+ calls)
- Practice discovery questions (MEDDPICC)

**Week 4+**:
- Start booking your own meetings
- Ask for feedback on every call

### For Hiring Managers

**Best Sources for Sales Talent**:
- Referrals from current team (offer $5K referral bonus)
- LinkedIn (search "Account Executive" + "SaaS" + "DevTools")
- Sales-specific job boards (RepVue, SalesJobs)
- Competitors (Datadog, HashiCorp, MongoDB, Snowflake)

**Interview Red Flags**:
- ‚ùå Can't explain past deals (MEDDPICC)
- ‚ùå Blames others for losses
- ‚ùå No questions about product/market
- ‚ùå Asks about comp before culture

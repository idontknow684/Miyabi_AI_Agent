### なぜYouTubeなのか？

**理由1: 日本市場での圧倒的リーチ**
- YouTube月間利用者: 7,000万人（日本）
- ゆっくり解説視聴者: 技術系に強い
- 開発者層への浸透率が高い

**理由2: 複雑な技術を分かりやすく**
- 数式・理論 → ゆっくりキャラで親しみやすく
- tmux可視化 → 動画で動きを見せる
- 長尺（10分）で深い理解を促進

**理由3: SEO・バイラル性**
- 「AIエージェント 仕組み」で検索上位狙い
- Twitter/Reddit拡散でオーガニックトラフィック
- 関連動画でシリーズ展開

**理由4: 信頼性構築**
- 論文の「顔」としての動画
- 投資家へのピッチ材料
- 採用時のブランディング

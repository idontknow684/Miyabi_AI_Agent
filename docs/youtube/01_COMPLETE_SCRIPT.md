**魔理沙**: 2つ目。「どんな意図も解決可能」だという公理。

**魔理沙**: 3つ目。「どんなゴールも分解できる」。

```

**[画面: グラフ - 進捗が単調増加する様子]**

**魔理沙**: 4つ目が超重要！「進捗は必ず前進する」。

**魔理沙**: 正解！これがあるから、**必ずゴールに近づく**ことが保証されるんだぜ。

### Axiom 5: Convergence (収束性)

**魔理沙**: そして5つ目。「有限回で収束する」。

```
∃n*: d(W_{n*}, W_goal) < ε

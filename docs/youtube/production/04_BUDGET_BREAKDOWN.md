## Cost-Saving Tips

### 1. Free Alternatives
- Use GIMP instead of Photoshop
- Use DaVinci Resolve instead of Premiere Pro (free version is powerful)
- Use Canva Free instead of Pro (limitations, but doable)
- Use free stock image sites (Unsplash, Pexels)
- Use YouTube Audio Library for BGM

### 2. Freelancer Platforms
- **Fiverr**: Cheap freelancers ($5-$50)
  - Thumbnail design: $25-$100
  - Script editing: $20-$50
  - Voiceover (if not using VOICEVOX): $50-$200
- **Upwork**: Higher quality ($20-$100/hour)
- **99designs**: Design contests ($299-$1,299)

### 3. Student/Startup Discounts
- Adobe Creative Cloud: 60% off for students
- Canva: Canva for Education (free for students)
- Many SaaS tools: Free or discounted for startups

### 4. Bundle Deals
- Adobe Creative Cloud (all apps): $54.99/mo vs. individual apps
- Epidemic Sound (unlimited music): $15/mo vs. per-track purchases

### 5. Timing
- Start free trials on Day 1 of production (not before)
- Cancel before renewal if not needed long-term
- Use monthly subscriptions, not annual

### 6. Reusable Assets
- Create asset library for future videos
- Invest in perpetual licenses (fonts, icons) vs. subscriptions
- Build relationships with freelancers (bulk discounts)

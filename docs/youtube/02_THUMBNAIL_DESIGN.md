## 実装手順（Photoshop / GIMP）

### Step 1: 背景作成
1. 新規キャンバス 1280×720px
2. グラデーション塗りつぶし（#0a1929 → #1a2f4a）
3. フィルター > ノイズ追加（5%）
4. 六角形パターンオーバーレイ（不透明度20%）

### Step 2: メインテキスト配置
1. テキストレイヤー作成
2. 「【衝撃】世界初！」を配置（中央上、赤色）
3. レイヤースタイル:
   - 境界線: 8px、黄色（#FFD700）
   - ドロップシャドウ: 4px、黒、不透明度70%
4. 「AIエージェントが数学的に証明された！？」を配置
   - 白縁取り 4px
   - グロー効果（外側、青白、10px）

### Step 3: 数式配置
1. 数式をTeX表記で作成（別ツール）
2. PNG透過で書き出し
3. 配置（中央）
4. レイヤースタイル:
   - 光彩（外側）: 20px、#AADDFF、ぼかし30px
   - 枠: 3px、#0099FF、グロー効果

### Step 4: キャラクター配置
1. 霊夢素材を左上に配置（200×250px）
2. 魔理沙素材を右上に配置
3. 境界線（白、2px）で縁取り
4. ドロップシャドウ追加

### Step 5: バッジ作成
1. 角丸長方形（180×100px、半径10px）
2. グラデーション（半透明白）
3. テキスト配置:
   - 上段: 「成功率」（24pt、白）
   - 下段: 「94.7%」（36pt、金色）
4. 3つコピーして配置

### Step 6: 仕上げ
1. 全体の色調整（彩度+10、明度+5）
2. シャープネス適用（0.5）
3. 最終確認
4. PNG書き出し（品質: 最高）

## モバイル対応チェック

**スマホプレビュー（360×202px）**:
- タイトルテキスト読めるか？
- 数式が潰れていないか？
- バッジの数字が見えるか？
- キャラクターが認識できるか？

**最小視認性基準**:
- メインテキスト: 最小36pt相当
- 数字: 最小24pt相当
- コントラスト比: 最低4.5:1
